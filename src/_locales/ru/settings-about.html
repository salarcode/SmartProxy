SmartProxy

Использование прокси с помощью SmartProxy не может быть проще. Считайте, что это решение — один раз настроить и забыть. Благодаря SmartProxy вам не нужно вручную менять прокси-сервер или включать/отключать его. Всё, что вам нужно сделать, это одним щелчком мыши сообщить SmartProxy, что вы хотите включить прокси для нужного сайта. После этого, когда вы в дальнейшем посетите этот конкретный сайт, SmartProxy подключится, и все ваши данные будут переданы на сайт через выбранный прокси без каких-либо действий с вашей стороны.

SmartProxy предоставляет пользователям от начинающих до опытных ряд функций, некоторые из которых перечислены здесь:

  • Включите прокси на вкладке, и для всего, что в ней открыто, прокси будет включён автоматически. Эта функция называется «Прокси для вкладки».
  • Автоматическое определение необходимости включения или отключения прокси для определённых сайтов на основе шаблонов правил
  • Простое переключение активного прокси их множества доступных
  • Включение прокси для текущего сайта одним щелчком мыши
  • Просмотр текущих элементов и запросов сайта для принятия решения о необходимости включения прокси одним щелчком мыши
  • Простое переключение между режимами прокси или включение прокси для всех доменов
  • Резервное копирование/восстановление настроек и правил

Обратите внимание:

SmartProxy не предоставляет никаких типов прокси-серверов.
SmartProxy никак вас не отслеживает.
SmartProxy полностью без рекламы.

Условные обозначения значков на панели инструментов

Значок на панели инструментов вашего браузера меняется, чтобы отображать статус SmartProxy. Вот что означают различные значки:

Условные обозначения значков (все браузеры):
Значок Значение
Запросы не будут использовать прокси.
Прокси применялся или будет применяться к запросам, соответствующим заданным правилам.
Или не соответствует правилу для вкладки.
Профиль: Smart Rules
Прокси применяется ко всем запросам в текущей вкладке. (В браузерах на основе Chrome есть домены, внесённые в белый список)
Профиль: Всегда включён
Решение об использовании любого прокси передано операционной системе.
Профиль: Системный
Условные обозначения значков (только для Firefox):
Значок Значение
Текущей вкладке сопоставлено правило. Все последующие запросы проходят через прокси.
Профиль: Умные правила
Текущей вкладке сопоставлено правило. Существует по крайней мере один неотвечающий/неудачный запрос.
Профиль: Умные правила
Текущей вкладке сопоставлено правило. По крайней мере один последующий запрос обошёл прокси с помощью правила.
Профиль: Умные правила
Текущей вкладке сопоставлено правило. По крайней мере один последующий запрос не соответствует ни одному правилу и не прошёл через прокси.
Профиль: Умные правила
Текущей вкладке не сопоставлено правило. По крайней мере один последующий запрос соответствует правилу и прошёл через прокси.
Профиль: Умные правила
Прокси применяется ко всем запросам в текущей вкладке. По крайней мере один последующий запрос обошёл прокси с помощью правила.
