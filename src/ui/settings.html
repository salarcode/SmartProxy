SmartProxy

Using proxies cannot be easier with SmartProxy. Think of it as configure once and forget solution. With SmartProxy you don't need to change your proxy manually or turn it on and off. All you have to do is to tell SmartProxy that you want to enable proxy for your desired website with just one click. After that, when you visit that specific website, SmartProxy will act and all you data for that website will be transferred though the proxy, without you doing anything.

SmartProxy brings a handful of features to users from beginner one to advanced one which some is listed here:

  • Just enable proxy for one site on a tab and everything there will have proxy enabled automatically. This feature is called proxy-per-tab
  • Automatically detect when to enable or disable proxy to certain websites based on rule patterns
  • Easily switch between many proxy servers as your active proxy server setting
  • Switch on proxy for current site you are on it with one click
  • View current website items and requests and decide whether to enable proxy for them with one click
  • Easily switch between proxy modes or enable proxy for all domains
  • Backup/Restore settings and rules

Please note that:

SmartProxy does not provide any type of proxy servers.
SmartProxy does not track you in any way.
SmartProxy is completely ad free in any way.

Welcome to SmartProxy

Welcome! I'm glad that you have chosen SmartProxy, the smart proxy switcher.
Please follow these steps to get started:

  1. Add a Proxy Server
    (Head to Proxy Servers tab and add a proxy server there)
  2. Set proxy mode to Smart Proxy mode
    (Use toolbar popup menu to change the active mode to Smart Proxy)
  3. Define Proxy Rules
    (Use toolbar popup menu to add rules or add/modify rules from Proxy Rules tab)
  4. You're all set.
